java项目开发中常见的路径详解

在讲解路径前，明确几个要点：

  • 项目中“/”表示当前路径。
  • 对于游览器访问的路径，“/”表示tomcat下的webapps目录。
  • 对于服务器发起的路劲，“/”表示项目的webRoot目录（IDEA中叫webApp）。
  • 对于项目src目录下的资源，项目部署后都会自动放在“/WEB-INF/classes”目录下。

###4、java中windows和lunix下路径差异解析
1、windows与lunix路径的不同之处

  • windows路径有盘符，所以c:\\web在Lunix下路径为/web
  • windows下的路径分隔符为”\”，而Lunix下为”/“。
  • windows下的路径要区分大小写。

2、差异解决方案：尽量用java提供的路径获取机制来期待路径常量
